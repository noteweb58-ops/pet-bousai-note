# 無料公開・収益化チェックリスト

## 1. ASP登録

- A8.netに無料登録する。
- バリューコマースに無料登録する。
- 楽天アフィリエイトに登録する。
- もしもアフィリエイトを使う場合は、Amazon・楽天・Yahoo案件に提携申請する。

## 2. リンク設置

- `affiliate-config.js` の検索URLを取得した広告URLに差し替える。
- トップページの購入候補リンクをクリックし、ASP管理画面でクリック計測されるか確認する。
- 売り切れや怪しい販売元に直行するリンクは避ける。

## 3. 無料公開

- GitHubでリポジトリを作成する。
- このフォルダのファイルをアップロードする。
- GitHub Pagesを有効化する。
- 公開URLでトップ、犬、猫、多頭飼い、法務ページを確認する。
- `node tools/configure-site.mjs --url 公開URL --email 連絡先メール` を実行する。

## 4. SEO初期設定

- `robots.txt` のURLを公開URLに差し替える。
- `sitemap.xml` のURLを公開URLに差し替える。
- Google Search Consoleに登録する。
- Bing Webmaster Toolsに登録する。
- サイトマップを送信する。

## 5. 月1回運用

- Search Consoleで表示回数が出ているキーワードを確認する。
- クリック率が低いページのタイトルと説明文を調整する。
- ASP管理画面でクリックが多い商品カテゴリを確認する。
- 成果が出ないリンクは、別ASPや別カテゴリへ差し替える。
- 犬、猫、多頭飼いの個別ページを1本ずつ増やす。
